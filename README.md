# ECASI AFRICA | Environmental Capacities and Sustainability Institute

Welcome to the official repository for the ECASI AFRICA (Environmental Capacities and Sustainability Institute) website. ECAS Institute is an independent Pan-African think tank advancing sustainable development through research, policy advisory, technical assistance, consultancy, and capacity strengthening.

## Project Overview

This project serves as the digital interface for ECASI AFRICA, providing a comprehensive platform to showcase our initiatives, resources, and impact across the African continent.

Our core focus areas are designed around the Data-Capacity-Planning Nexus, and the platform provides access to:

- **Thematic Specialties:** Information on programs such as the Clean Air Programme, Energy, Sustainable Agriculture and Food Systems, Transport, Waste and Circular Economy, Forests, Mining & Extractives, Oceans & Marine, Security & Migration, and Water.
- **Programs & Capacity Building:** Details on Training & Education, Research & Systematic Observation, Creation of Enabling Environments, Adaptation & Mitigation Assessments, GHG Inventories, Institutional Capacity Building, Climate Change Programming, National Communications, and Policy Analysis.
- **Knowledge Resources:** A comprehensive library of Publications, Policies & Laws, Reports, Videos, Books, and Policy Briefs.
- **Institutional Information:** Insights into our History, Strategic Focus, Governance, Governing Council, Theory of Change, and our Team.
- **Updates & Media:** A Newsroom and Gallery to keep stakeholders informed of our latest activities and events.

## Technologies Used

This project is built using a modern, robust front-end stack to ensure performance, accessibility, and a seamless user experience:

- **React (v18):** For building a dynamic, component-driven user interface.
- **Vite:** Next-generation frontend tooling for fast development and optimized production builds.
- **Tailwind CSS:** Utility-first CSS framework for rapid and responsive styling.
- **Shadcn UI & Radix UI:** For accessible, customizable, and high-quality UI components.
- **React Router (v6):** For seamless client-side routing and navigation.
- **Framer Motion:** For sophisticated animations and transitions.
- **React Query:** For efficient data fetching and state management.

## Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites
- Node.js installed on your system.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd ecasi_africa_
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```
4.  **View the application:**
    Open your browser and navigate to the local URL provided in your terminal (typically `http://localhost:5173`).

## Building for Production

To create a production-ready build:
```bash
npm run build
```
This will generate optimized static assets in the `dist` directory.
